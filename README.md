<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Python 3.9
* Pip
* Virtualenv
* Git
* gTTS
* SpeechRecognition
* playsound
* webbrowser

### Installation

1. Clone the repo
   ```sh
    git clone git@github.com:josafamarengo/virtual-assistant.git
   ```
2. Install Python packages
   ```sh
    pip install -r requirements.txt
    ```
   
3. Run the program
4. Enjoy!
